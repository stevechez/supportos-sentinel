# Value Evidence Log (Phase 20E)

The move from "customers like it" to "customers received measurable
value" happens one real quote at a time, in this document. No invented ROI
numbers, no rounding up -- if a customer says something vague, log it as
vague. This log is the raw material `docs/customer/case-study-template.md`
gets filled in from later.

## How to log an entry

One entry per concrete before/after pair, in the customer's own words
where possible. Add a new entry at the top each time.

## Value signal categories

- **Time saved** -- a task that used to take manual effort now doesn't.
- **Issues discovered** -- a pattern the customer didn't know about before.
- **Documentation improved** -- a specific doc/macro/process that changed
  because of a Sentinel finding.
- **Escalations reduced** -- fewer tickets needing a human because a
  self-service gap got fixed.
- **Faster response** -- time-to-resolution improved for a specific issue
  type.
