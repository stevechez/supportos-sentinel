# Follow-Up Email Templates

Short, plain, no marketing voice -- these are from a founder to one real
person they just talked to, not a drip campaign.

## Day 0 -- right after the onboarding call

Subject: Your Sentinel workspace

```
Hi [name],

Good talking today. Your workspace is set up and connected to [their
tool] -- you should already be seeing your first insight.

Two things:
1. If anything is confusing, click the Feedback button in the
   bottom-right of any page -- I read every one myself.
2. I'll check in around [date, ~day 7] to see what you're finding.

Talk soon,
[founder name]
```

## Day 3 -- only if they haven't connected a source yet

Subject: Quick check-in

```
Hi [name],

Noticed you haven't connected [their tool] yet -- totally fine if you've
just been busy, but wanted to check if anything got in the way. It's a
single click on your dashboard, and I'm happy to hop on a 5-minute call
if that's easier.

[founder name]
```

## Day 7 -- check-in

Subject: What's Sentinel finding for you?

```
Hi [name],

Quick check-in: what has Sentinel surfaced so far, and did anything
prompt you to actually change something (a doc, a macro, a process)?

If nothing's jumped out yet, that's useful to know too -- tell me what
you expected to see that you haven't.

[founder name]
```

## Day 30 -- evaluation

Subject: 30 days in -- can we talk for 15 minutes?

```
Hi [name],

You've been using Sentinel for about a month now. I'd love 15 minutes to
hear how it's gone -- what worked, what didn't, and whether this is
something you'd want to keep using.

[calendar link]

[founder name]
```

## If a pilot has gone quiet

Subject: Still here if useful

```
Hi [name],

Haven't heard from you in a bit -- no pressure at all, just wanted to
check whether Sentinel wasn't useful, wasn't the right time, or something
else got in the way. Honestly curious either way, and it helps me
improve the product if you're willing to share.

[founder name]
```
